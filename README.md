# neoride-admin
Admin Panel for managing all core data and operations of the mobile application.
Built with Angular, this dashboard provides internal tools for monitoring, controlling, and configuring the platform.

Key Features

User Management: View, filter, edit, block/unblock users.

Ride/Traffic Monitoring: Track live and historical activity, usage metrics, and operational status.

Payments & Transactions: Review transactions, verify payment issues, handle refunds, and audit logs.

Service Configurations: Manage app settings, pricing, offers, and operational rules.

Content & Data Control: Handle categories, assets, notifications, and static content updates.

Analytics Overview: High-level insights into platform performance.

Tech Stack

Angular – Frontend framework for building scalable and modular admin interface.

REST API – Primary communication layer for fetching and updating application data.

Socket.IO – Real-time updates for live metrics, traffic monitoring, and instant data sync.

Bootstrap – Base styling and responsive layout utilities.

CoreUI – Component library and admin UI framework for consistent, production-ready dashboard design.
